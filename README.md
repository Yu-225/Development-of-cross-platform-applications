# Development-of-cross-platform-applications

Цей репозиторій збирає практичні лабораторні роботи з побудови кросплатформених GUI на Tkinter: кожна папка `LabN` демонструє окрему тему та ілюструє роботу з моделлю, виглядом і контролером.

## Актуальні лабораторні
1. **Lab1** – проста MVC-структура з `ModelApp`, `ViewApp` і `ControllerApp`, де модель тримає список напрямів компаса, а кнопки Up/Down/Reset змінюють індекс і оновлюють мітку.
2. **Lab2** – цифрова клавіатура з PIN-кодом: комбіновані `Button`-и викликають `ModelApp` (вставка/видалення символів, перевірка проти `8085`), а мітка змінює колір на зелений або червоний залежно від результату.
3. **Lab3** – демонстрація стилів і геометрій для `Label`: кнопки змінюють атрибути (`width`, `height`, `font`, `relief`, `highlight*` тощо) або застосовують попередньо визначені теми, а контролер підключає `LightTheme` і `DarkTheme`.
4. **Lab4** – опитування про мобільні застосунки: `components.py` визначає базові обгортки для `Combobox`, `Radiobutton`, `Checkbutton`, `Spinbox`, `Scale`, `ViewApp` додає кожний компонент до форми, а `ControllerApp` збирає стан і відображає його в табличному вигляді з кнопкою очистки.

## Запуск
Кожна лабораторна — самостійний скрипт. Наприклад:
```
python Lab1/main.py
python Lab2/main.py
python Lab3/main.py
python Lab4/main.py
```
